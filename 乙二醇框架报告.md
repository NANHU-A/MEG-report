2. 乙二醇品种情况介绍
基础属性：
乙二醇是无色、低粘度、带甜味的二元醇，工业上最重要的身份不是“化学品本身”，而是聚酯产业链的关键原料。

主要生产路线：
1. 石油路线：以乙烯氧化制环氧乙烷，再水合制乙二醇，工艺成熟、规模化程度高。
2. 煤制路线：以煤为起点，经合成气、草酸酯等中间路径制取乙二醇，受煤价和工艺效率影响更大。
3. 天然气路线：海外部分地区可利用气化资源优势，成本与原料气价格强相关。

品种特点：
1. 供给端“重装置、重资本、重检修”。
2. 需求端“重聚酯、重终端消费”。
3. 价格上容易出现“原料强、需求弱”或“需求强、供应紧”的阶段性错配。

小白理解法：
可以把乙二醇理解成“做衣服、做瓶子、做包装的上游中间材料”，上游看油煤气，中游看工厂开工，下游看聚酯和消费。

2. 乙二醇的工艺与成本逻辑
石脑油/乙烯路线
优势：技术成熟、配套聚烯烃和烯烃装置多
劣势：对原油价格敏感，利润受炼化一体化影响大

煤制乙二醇路线
优势：煤炭资源丰富地区可形成原料优势
劣势：工艺链条长、能耗和环保要求高、开工容易受利润挤压

天然气/合成气路线
优势：原料气资源便利时成本稳定性较好
劣势：资源禀赋差异大，区域性很强

成本观察要点：
1. 不同工艺的成本曲线差异，决定了行业利润分布。
2. 行业利润改善时，边际装置复产和高负荷运行往往会增加供给。
3. 行业利润恶化时，落后产能和高成本装置更容易降负荷或停车。

3.1 全球供应: 乙二醇供应结构
全球乙二醇供应长期呈现“中东、北美、亚洲”多中心格局，其中中东依托能源和港口优势，适合做出口型供应；北美具备原料与化工配套优势；亚洲则兼具生产、消费和贸易三重属性。

供应格局的核心特点：
1. 海外装置多集中在大炼化或大型综合化工园区，单装置规模较大。
2. 中东和北美的出口能力较强，对亚洲价格影响明显。
3. 海运链条决定了区域间价差，远洋运费、船期和国际套利窗口会改变到港节奏。

影响全球供应的因素：
1. 原料价格波动。
2. 计划检修和非计划停车。
3. 国际贸易环境和运费变化。
4. 区域套利窗口变化。

3.2 全球需求: 乙二醇需求结构
乙二醇的全球需求高度集中在聚酯产业链，尤其是纤维级消费最为重要。

主要需求去向：
1. 涤纶长丝：服装、家纺、工业纺织等领域，需求与纺织订单、出口景气相关。
2. 涤纶短纤：更偏功能性和替代性消费，受终端制造业影响。
3. PET瓶片：饮料、食品包装、日化包装等，受消费和包装需求驱动。

需求端的观察逻辑：
1. 先看聚酯开工，再看终端订单。
2. 终端好，聚酯敢开工，乙二醇就容易去库。
3. 终端差，聚酯主动降负荷，乙二醇库存会快速累积。

一句话理解：
乙二醇不是“自己创造需求”的品种，它的需求基本由聚酯和消费链条代为决定。

本章小结：需求端的核心变量是聚酯开工，终端订单决定需求持续性。

3.2 全球供需平衡与物流
全球乙二醇的贸易流向，本质上是“低成本产地向高消费区域输送”。

主要物流方向：
1. 中东 → 亚洲：最核心的远洋流向。
2. 北美 → 亚洲：在套利窗口打开时更活跃。
3. 亚洲内部：区域间调货更多体现为港口库存和船期变化。

定性判断要点：
1. 海外装置集中检修时，亚洲进口到港会阶段性下降。
2. 当国际价差扩大，进口积极性提高，国内库存容易抬升。
3. 当聚酯旺季到来而到港偏少，现货会明显强于期货。

市场含义：
物流不是单纯的运输问题，而是区域价差的结果。价差扩大，货才会动；价差收窄，货流就会放缓。

4.1 国内供应: 国产及进口
国内乙二醇供应通常由国产与进口共同构成，其中进口是影响短期边际的关键变量。

国产供应：
1. 国内装置以大型化工和煤化工装置为主。
2. 国产供给受利润、检修和开工率影响较大。
3. 供给扩张期往往伴随新增产能投放和行业竞争加剧。

进口供应：
1. 进口来自海外大装置，具有船期和到港节奏特征。
2. 进口量变化直接影响港口库存和现货压力。
3. 若海外价格低于国内，进口通常更有动力；反之则进口积极性下降。

国内供应的观察重点：
不是只看“有没有产能”，而是看“实际开工了多少、进口到了多少、港口压了多少”。

4.2 国内需求: 聚酯消费为核心
国内乙二醇几乎是“看聚酯脸色”的品种。

主要消费方向：
1. 涤纶长丝：占比最大，和服装、家纺、工业丝联系紧密。
2. 涤纶短纤：和替代消费、无纺布、填充材料相关。
3. PET瓶片：与饮料、包装和消费升级相关。

国内需求特点：
1. 季节性明显，传统上旺季和淡季切换会影响聚酯负荷。
2. 需求并不总是线性增长，更多表现为阶段性补库和去库。
3. 下游利润会倒逼上游，聚酯利润差时，乙二醇需求会被动下降。

怎么看需求强弱：
1. 看聚酯工厂开工有没有持续上升。
2. 看终端订单和织造、包装行业景气。
3. 看库存是主动去化还是被动累积。

4.3 国内年度平衡及库存
乙二醇的国内库存，是判断供需松紧最直观的指标之一。

库存的含义：
1. 库存上升，说明供给偏多或需求偏弱。
2. 库存下降，说明供给收缩或需求回暖。
3. 库存变化往往领先价格拐点，但不会单独决定价格。

定性框架：
进口到港增加 + 国产开工提升 + 聚酯开工走弱 = 库存上行压力
进口到港减少 + 国产检修增多 + 聚酯补库 = 库存去化窗口

库存观察要点：
1. 不只看绝对水平，更要看连续变化方向。
2. 港口库存比厂内库存更敏感，往往更早反映贸易流向。
3. 当库存和价格背离时，往往意味着市场在交易预期而不是现实。

5. 价格价差
乙二醇价格分析，核心不是单点价格，而是“原料 - 生产 - 现货 - 期货 - 下游利润”整条链。

1. 利润
上游原料涨跌快，下游聚酯利润变化慢，容易出现“成本推升但需求跟不上”的阶段。
装置利润是判断供应扩张还是收缩的重要信号。

2. 基差
基差反映现货与盘面之间的偏离程度，最能体现短期供需和库存压力。
现货偏紧时，基差往往偏强；库存偏高时，基差容易走弱。

3. 月差
月差体现近远月供需预期差。
近月紧、远月松时，正套机会更明显；反之则可能反映远月供应改善。

4. 品种价差
乙二醇与原油、煤、石脑油、PTA、PX等链条品种之间，存在明显的传导和替代关系。
更重要的是看聚酯链利润分配，而不是孤立看乙二醇单品。

5. 区域价差
国内外价差决定进口积极性，港口和内地价差决定国内流通。

5. 价格驱动的典型场景
场景一：原料上涨但需求偏弱
乙二醇成本被推高，但若聚酯开工跟不上，价格可能冲高后回落，利润先被压缩。

场景二：检修集中 + 库存下降
供应端突然收缩，叠加聚酯补库，现货和近月合约通常更强。

场景三：进口增加 + 终端走弱
到港抬升压港口库存，现货承压，盘面往往先交易弱预期。

场景四：旺季预期兑现
若终端订单改善，聚酯开工抬升，库存去化，乙二醇可能从“看空逻辑”转向“修复逻辑”。

交易理解：
乙二醇不是“单边永远跟成本走”的品种，而是一个强烈受库存、开工和预期影响的高弹性化工品。

本章小结：短期交易更应关注库存、开工和进口节奏，成本仅提供下限参考。

6. 交易标的
乙二醇期货是把现货产业链的价格波动转化为可交易工具的核心载体。

交易意义：
1. 现货企业可用于套期保值。
2. 产业链参与者可用于锁定采购或销售利润。
3. 研究者可通过期货观察市场预期变化。

常见关注点：
1. 交割标准与品质要求。
2. 合约月份对应的季节性和库存节奏。
3. 期现联动和基差变化。

对小白来说：
期货不是为了猜价格，而是为了管理价格风险。对乙二醇这种强周期、大波动的化工品，期货的意义尤其大。

本章小结：交易标的是产业定价的延伸工具，核心价值在于风险管理而非单纯投机。

本章小结：研究路径应遵循“成本-供给-需求-库存-价差-预期”的顺序，避免只盯价格。

8. 风险提示
1. 原料价格大幅波动可能导致成本端快速重估。
2. 装置集中检修、事故或不可抗力会改变短期供给。
3. 进口船期、海运和国际贸易环境会影响到港节奏。
4. 下游聚酯开工若不及预期，需求修复可能被证伪。
5. 市场交易逻辑切换很快，库存和基差信号需要结合看，不能只看单一指标。

本章小结：乙二醇属于高波动化工品，研究和交易都要预留库存、成本和需求预期反转的风险。
