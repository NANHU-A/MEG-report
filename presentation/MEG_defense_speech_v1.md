# MEG答辩逐页讲稿（v1）

## 使用说明

- 适用场景：产业+交易型评委联合答辩。
- 讲述主线：供给 -> 需求 -> 物流 -> 库存 -> 利润；价格仅作为传导结果解释，不做点位预测。
- 覆盖方式：采用覆盖模式B。实质内容页详讲；章节分隔页与仅数据来源页给过渡句。
- 结论风格：稳健中性，以条件触发、证伪条件、风险提示为核心。

## 全局时间分配

- 总时长目标：约42分钟（可在35-42分钟内伸缩）。
- 一级重点页（14页）：约23分50秒。
- 二级重点页（15页）：约11分45秒。
- 三级串场页（其余页）：约6分25秒。
- 机动与互动缓冲：按现场提问弹性压缩，默认不单独占时。

## 价格传导逻辑说明（全场统一口径）

在本次答辩中，价格不是单独预测对象，而是链条末端变量。
原料与进口成本变化，先影响到港节奏和港口库存。
随后通过现货紧张度改变产业利润分配，最后传导到盘面价格与价差结构。
我们只做方向和节奏判断，不做绝对点位判断。

## 逐页讲稿

### Page 2（一级重点，建议2分30秒）
- 页面定位：全篇总纲与研究方法页。
- 台上讲稿：这一页我先把框架讲清楚，避免后面陷入“只看价格”的误区。我们本次研究按五步走：先看供给，再看需求，再看物流，再看库存，最后落到利润；价格放在最后解释，不做先验押注。对产业端，这个框架能回答“开不开、采不采、卖不卖”；对交易端，这个框架能回答“什么时候做、做到哪一步、什么条件下退出”。核心变量就四个：装置开工、聚酯负荷、港口库存、利润变化。后面每一页我都按“发生了什么、为什么、对库存和利润意味着什么、交易上怎么做”来讲，确保产业和交易两类评委都能直接落地。
- 一句话结论：先判供需与库存方向，再看利润和节奏，价格只是结果映射。
- 若被追问：
  - 问：为什么不直接预测价格区间？
  - 答：因为价格是多变量合成结果，先抓供需库存与利润，胜率和稳定性更高。
  - 问：这套框架最大的好处是什么？
  - 答：同一套口径同时服务生产决策和交易决策，减少“看对逻辑却做错节奏”。

### Page 3（三级串场，建议20秒）
- 页面定位：章节分隔页（产业结构）。
- 台上讲稿：下面进入产业结构部分，我会先把MEG在上下游链条中的位置讲清楚，再说明哪些环节对后续供需和利润最敏感。
- 一句话结论：先把链条看懂，后面所有变量才有坐标系。
- 若被追问：
  - 问：这一章最关注什么？
  - 答：关注成本来源、贸易属性和需求集中度三件事。

### Page 4（一级重点，建议1分40秒）
- 页面定位：产业链结构与上下游约束页。
- 台上讲稿：MEG是典型的中游大宗，最上游受油、气、煤成本约束，下游高度绑定聚酯。这里最关键的不是“用途多不多”，而是“需求集中不集中”。从结构上看，聚酯相关消费占绝对主体，意味着只要聚酯负荷发生边际变化，MEG平衡表就会被快速改写。第二个点是贸易属性强。中东、北美凭低成本路线形成外部供给锚，中国虽然产能大，但并没有完全脱离进口约束。第三个点是国内煤制路线形成中国特色弹性，能够在利润修复时快速提负，但成本和副产物约束决定了其不是无上限扩张。对交易来说，这一页要记住：MEG不是单一能源品，也不是单一化工品，而是“成本+贸易+聚酯需求”三重驱动。
- 一句话结论：MEG的本质是高贸易属性的聚酯原料，需求集中决定其弹性大。
- 若被追问：
  - 问：为什么强调“贸易属性”而不是“国内产能”？
  - 答：因为边际波动主要由进口到港和港口库存触发，国内产能更多决定中期底盘。

### Page 5（二级重点，建议45秒）
- 页面定位：工艺路线对成本与弹性的解释页。
- 台上讲稿：工艺路线决定成本曲线位置，也决定开工弹性。油制受原油与石脑油波动影响更直接；气制在中东、北美有稳定低成本优势；煤制在中国有补位能力，但对煤价、装置效率和环保约束更敏感。后续判断利润时，要把不同工艺放在同一利润尺子上看边际开停。
- 一句话结论：路线不同，成本不同，开工反馈速度也不同。
- 若被追问：
  - 问：哪个路线最能决定价格中枢？
  - 答：通常看边际成本路线，但短期波动往往由进口与库存节奏放大。

### Page 6（三级串场，建议20秒）
- 页面定位：工艺补充页。
- 台上讲稿：这一页作为补充只强调一点：同样是新增产能，不同路线对有效供给贡献不同，不能只看名义产能。
- 一句话结论：名义扩产不等于有效供给同步增加。
- 若被追问：
  - 问：有效供给怎么判断？
  - 答：看负荷稳定性、现金流利润和检修兑现率。

### Page 7（三级串场，建议20秒）
- 页面定位：副产物影响页。
- 台上讲稿：副产物价格会反向影响MEG现金流，尤其在利润薄的阶段，副产物改善能推迟减产，副产物走弱会加速降负。
- 一句话结论：副产物是决定供给韧性的隐性杠杆。
- 若被追问：
  - 问：副产物影响大吗？
  - 答：在高利润期影响小，在盈亏平衡附近影响明显。

### Page 8（二级重点，建议45秒）
- 页面定位：价格传导机制页。
- 台上讲稿：这页是全场口径锚点。我们不是“先看价格再找理由”，而是先看成本、供需、物流、库存、利润，最后解释价格。若成本下移但库存不去，价格反弹通常不可持续；若成本不变但到港收缩、库存加速去化，价格也可能上行。交易上应盯传导链是否完整，而不是单看某一个信号。
- 一句话结论：只有传导链条完整，价格方向才有持续性。
- 若被追问：
  - 问：最容易误判的环节是什么？
  - 答：把成本变化当成单一驱动，忽视库存与利润反馈。

### Page 9（一级重点，建议1分30秒）
- 页面定位：供需指标体系页。
- 台上讲稿：这一页给出可执行的监控面板。供给端看装置开工与检修兑现，需求端看聚酯负荷与订单，物流端看到港与航线扰动，库存端看港口绝对量和周转天数，利润端看MEG与下游的利润分配。对产业端，建议按周跟踪边际变化；对交易端，建议按“信号共振”决策，至少三项同向再提高仓位。尤其注意库存信号，库存是验证前面所有假设的最终裁判。当前框架下，单点数据的解释力有限，组合信号更可靠。
- 一句话结论：交易不是比谁看得快，而是比谁验证得完整。
- 若被追问：
  - 问：最优先跟踪哪三个指标？
  - 答：聚酯负荷、进口到港、港口库存。

### Page 10（二级重点，建议50秒）
- 页面定位：上游原料成本背景页。
- 台上讲稿：2025年原油与石脑油整体重心下移，成本端对MEG的支撑较前期弱化。但这里要强调，成本下移只提供“下限参考”，不直接给出趋势。若同期出现到港收缩和库存去化，盘面仍可能偏强；反过来，成本稳住但库存持续累积，价格也会承压。我们把成本放在框架第一层，但绝不把它当唯一结论。
- 一句话结论：成本决定地板，库存和利润决定弹跳高度。
- 若被追问：
  - 问：成本端现在最该看什么？
  - 答：看油价波动是否引发不同工艺利润再平衡，而不只看绝对油价。

### Page 11（三级串场，建议20秒）
- 页面定位：章节分隔页（全球供需）。
- 台上讲稿：下面进入全球供需，重点看全球供给重心变化如何通过贸易流向影响中国库存节奏。
- 一句话结论：全球结构变化，最终都会回到中国港口库存上。
- 若被追问：
  - 问：全球部分最关键结论是什么？
  - 答：中国是产能中心，但并非绝对定价中心。

### Page 12（一级重点，建议1分40秒）
- 页面定位：全球供应地区结构页。
- 台上讲稿：全球MEG产能约6181万吨，中国占比约50%，中东约20%，北美约15%。表面看中国体量最大，但成本优势并不完全在中国。中东和北美的低成本气头路线，对全球边际供给有持续影响力。过去五年中国扩产很快，但开工率大致在65%-70%，说明“有产能不等于有有效供给”。这对交易的含义是：不能把国内新增产能直接等同于过剩，必须结合利润和开工兑现；同时外盘低成本货源若集中流入，会通过进口冲击更快影响国内库存与价差。
- 一句话结论：中国决定体量，中东和北美仍在决定边际成本与贸易压力。
- 若被追问：
  - 问：中国占一半产能为何还会被进口影响？
  - 答：因为边际供给看成本和航运可达性，低成本货源到港时对现货冲击更直接。

### Page 13（一级重点，建议1分30秒）
- 页面定位：全球工艺结构与成本分层页。
- 台上讲稿：全球工艺大致是油头45%、气头25%、煤头约5%（煤头主要在中国）。这张图真正要表达的是“成本分层”而非“工艺占比”。气头在多数阶段处于成本低位，油头受原油链条波动更大，煤头在国内具备调节能力。贸易流向长期从低成本区向高需求区流动，因此当海外气头负荷稳定且运费可控时，中国进口压力往往上升。反之，一旦地缘事件抬升航运不确定性，低成本优势会被物流摩擦部分抵消，国内供需就会重新收紧。
- 一句话结论：看懂工艺结构，本质是在看全球边际成本曲线。
- 若被追问：
  - 问：煤制在全球占比低，为什么还要重点看？
  - 答：因为它决定中国本土供给弹性，是国内平衡表的关键调节器。

### Page 14（二级重点，建议45秒）
- 页面定位：全球产能集中度与议价页。
- 台上讲稿：全球前五厂商占比高，说明大型装置对阶段性供应有较强影响。中国扩产把重心往东移，但集中度高意味着检修和重启行为仍可能造成阶段性波动。交易上要结合大厂检修窗口与进口船期，判断短期供给收缩是否真实。
- 一句话结论：集中度提高了“事件驱动”在短周期里的影响力。
- 若被追问：
  - 问：该怎么跟踪事件驱动？
  - 答：跟踪大厂检修计划、兑现率和替代货源到港时间差。

### Page 15（三级串场，建议20秒）
- 页面定位：全球下游补充页。
- 台上讲稿：这一页作为背景补充，只强调一点：下游聚酯产能分布变化会反向决定MEG的跨区域流向。
- 一句话结论：需求重心在哪，MEG贸易流就往哪走。
- 若被追问：
  - 问：是否会改变中国角色？
  - 答：短期不会，但会改变中国出口与进口的结构节奏。

### Page 16（二级重点，建议40秒）
- 页面定位：全球消费结构页。
- 台上讲稿：全球消费结构整体稳定，但区域增速差异明显。亚洲仍是需求核心，欧美更多体现结构性替代与再生材料扰动。对MEG而言，需求不是“突然消失”，而是“区域和节奏重分配”。因此我们对需求判断更看边际而非总量。
- 一句话结论：需求总盘稳中有变，真正变化在区域和节奏。
- 若被追问：
  - 问：需求端最大的误区是什么？
  - 答：把结构调整误判为总需求崩塌。

### Page 17（三级串场，建议25秒）
- 页面定位：全球瓶片需求变化页。
- 台上讲稿：瓶片需求仍有韧性，但贸易壁垒风险在上升，意味着未来出口导向型需求可能更受政策影响。
- 一句话结论：需求有韧性，但跨境贸易摩擦是中期变量。
- 若被追问：
  - 问：这会立即冲击MEG吗？
  - 答：短期影响有限，中期会通过聚酯开工间接传导。

### Page 18（二级重点，建议40秒）
- 页面定位：亚洲MEG消费变化页。
- 台上讲稿：亚洲消费决定全球边际需求，中国处于核心位置。若亚洲终端补库启动，MEG去库速度会明显加快；若补库不及预期，库存压力会在港口端优先体现。这里建议把亚洲消费与中国港口库存联动跟踪。
- 一句话结论：亚洲需求是决定中国库存方向的重要外部变量。
- 若被追问：
  - 问：联动指标怎么设？
  - 答：看亚洲下游开工边际变化与中国到港后库存去化速率是否同向。

### Page 19（二级重点，建议55秒）
- 页面定位：全球MEG物流网络页。
- 台上讲稿：物流是把“成本优势”变成“现货压力”的通道。中东到东北亚是关键航线，任何港口拥堵、运费波动、地缘扰动，都会改变到港节奏。对产业端，物流变化意味着采购与套保窗口变化；对交易端，物流变化意味着库存拐点可能提前或延后。重点不是事件本身，而是事件是否改变可交割资源进入国内市场的时间分布。
- 一句话结论：物流不改变总量，但会重排时间，进而重排价格节奏。
- 若被追问：
  - 问：物流信号如何确认有效？
  - 答：看船期变化能否在2-4周内映射到港口库存与基差。

### Page 20（三级串场，建议20秒）
- 页面定位：章节分隔页（中国供需）。
- 台上讲稿：下面进入国内供需，这是本次答辩权重最高的部分，尤其重点讲进口、库存和利润如何共同决定交易节奏。
- 一句话结论：国内章节决定“能不能做”和“什么时候做”。
- 若被追问：
  - 问：国内段最核心页是哪几页？
  - 答：24到27页、29到30页、41页。

### Page 21（一级重点，建议1分25秒）
- 页面定位：中国供给结构总览页。
- 台上讲稿：中国供给结构可以概括为“自产提升，但外部约束未消失”。自产占比提高后，国内对外依赖边际下降，但在边际成本和短期现货可得性上，进口仍具决定力。特别在旺季或装置检修期，进口节奏变化对港口库存影响更快。这里对交易的意义是，不能把“国产化提升”解读成“进口不重要”，二者是并行变量：前者决定中期韧性，后者决定短期波动。
- 一句话结论：供给安全边际在提升，但短期波动仍受进口节奏主导。
- 若被追问：
  - 问：供给判断先看什么？
  - 答：先看利润约束下的国内负荷，再看进口到港是否补位。

### Page 22（二级重点，建议45秒）
- 页面定位：中国自产能力页。
- 台上讲稿：自产约57.1%，说明国内已具备较强底盘。但“底盘强”不代表“波动小”。当利润压缩时，国内装置降负仍会出现，若同时进口不畅，库存可能快速去化；反过来，利润修复叠加进口回升，又可能形成再累库。供给判断必须放在利润与进口的联合框架下。
- 一句话结论：自产决定韧性，利润与进口决定波动幅度。
- 若被追问：
  - 问：自产提高是否降低交易机会？
  - 答：不会，机会从“总量短缺”转向“节奏错配”。

### Page 23（三级串场，建议25秒）
- 页面定位：聚酯链条区域结构补充页。
- 台上讲稿：下游产能区域分布变化会影响内贸流向和采购节奏，但对MEG的核心仍是聚酯负荷是否持续在高位。
- 一句话结论：区域变化影响流向，负荷变化决定强弱。
- 若被追问：
  - 问：区域分散会怎样？
  - 答：会提高物流与仓储管理难度，但不改变需求主逻辑。

- 一句话结论：交易胜率来自条件组合，不来自单一观点。

## Q&A备答清单

1. 问：为什么把价格放在最后讲？  
   答：价格是结果变量，先看供需、物流、库存、利润，判断更稳定、可验证。

2. 问：中国产能占比高，为什么还受外盘影响？  
   答：边际供给仍受低成本进口影响，尤其在到港节奏变化时，库存反应更快。

3. 问：进口依存度下降后，地缘风险是不是不重要了？  
   答：不对。总量依存下降但来源结构仍集中，航线扰动仍会放大短期波动。

4. 问：如何判断去库是否可持续？  
   答：看“到港放缓+库存连续下降+利润修复”三项是否持续共振。

5. 问：如果成本下行，MEG一定走弱吗？  
   答：不一定。若同步出现到港收缩和需求稳住，价格仍可能阶段偏强。

6. 问：聚酯负荷和聚酯利润哪个更重要？  
   答：负荷决定当期需求强度，利润决定需求持续性，两者需联合判断。

7. 问：库存高位时能做多吗？  
   答：可以做反弹，但应定义为交易性仓位，等待库存拐点后再升为趋势仓位。

8. 问：出口数据改善能否替代进口逻辑？  
   答：短期不能。当前中国MEG仍是进口调节主导，出口更多是边际补充。

9. 问：交易上最关键的证伪条件是什么？  
   答：库存方向反向持续两周以上，说明主假设失效，应立即降仓。

10. 问：产业企业如何落地这套框架？  
    答：把库存目标和利润阈值前置，按触发条件动态调整采购与套保比例。

11. 问：若出现突发地缘冲突，第一步看什么？  
    答：先看船期和到港预报是否实质下修，再看库存是否同步去化。

12. 问：当前更偏趋势还是区间？  
    答：在库存未形成持续单边前，更偏区间节奏；趋势需等待信号共振确认。

13. 问：如何避免“看对逻辑做错交易”？  
    答：把观点拆成触发、证伪、风险三段，按规则执行而不是按情绪加仓。

14. 问：评估一轮上涨质量的核心指标是什么？  
    答：库存去化斜率与利润修复同步性，二者同向才是高质量上涨。

15. 问：一句话给本轮MEG研究定位？  
    答：这是“条件触发型”研究，不是“点位预测型”研究。
